📦 Delivery Logistics Backend
A scalable backend system built with NestJS, PostgreSQL, RabbitMQ, and Docker. Supports JWT authentication, role-based access control, and driver/order/address management.

🚀 Tech Stack
Framework: NestJS

Database: PostgreSQL

Message Queue: RabbitMQ

Auth: JWT

API Docs: Swagger (in progress)

Containerization: Docker

🔐 Features
JWT authentication system (sign up / login)

Role-based access: admin, driver, client

Order lifecycle management

Driver management

Each user can have multiple saved addresses

Task queuing with RabbitMQ

Swagger documentation (WIP)
